# Eighth Artifact Upload Policy

CI parity jobs should upload a single parity artifact bundle per surface:

- `parity-snapshot-artifacts`
- `parity-replay-artifacts`
- `parity-visual-docs-artifacts`

Bundle expectations:

- include benchmark snapshot outputs when available
- include replay storage outputs when available
- include release/signoff/handoff docs when available
- always include `MANIFEST.md`
