# Round 5 Operator Quickstart Refresh

1. Start with `/status`, `/brief`, and `/inspect artifact latest-orchestration`.
2. Use `/inspect workflows latest` and `/coordinate latest` to review the newest execution artifacts before re-running anything.
3. Use `/inspect artifact bundle` after `/export diagnostics` to reopen the latest workspace index quickly.
4. If an artifact shows `warm` or `stale`, follow the suggested refresh commands from the inspector footer before trusting it.
