# Round 9 Sub-Agent Team Quickstart

1. 用 `team_create` 建立 team runtime，明确 `goal`、成员和 permission inheritance。
2. 用 `send_message` 给成员补充任务、handoff 或 follow-up。
3. 用 `team_monitor` 或 `/inspect artifact latest-agent-team-monitor` 查看团队状态。
4. 用 `/tasks monitor` 和 `/inspect artifact latest-subagent-result` 跟进后台结果。
5. 对需要继续追踪的成员结果，回到 parent session 做 review / merge / next-step。
