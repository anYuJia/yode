# Yode Desktop

Yode desktop preview app for the first Tauri migration batch.

```bash
pnpm install
pnpm dev
pnpm tauri:dev
```

Current scope:

- Tauri v2 shell.
- React + TypeScript + Vite frontend.
- Tailwind v4 CSS entry.
- Mock sidebar, topbar, timeline, composer and settings shell.
- No real AgentEngine bridge yet.
